# Imouse Clicker v1.0.0

First public release of Imouse Clicker for Apple silicon Macs.

## Features

- Left-side tap to primary click
- Right-side tap to context menu
- Optional double-tap to double-click
- Native macOS menu-bar controls
- Accessibility permission handling
- Single-instance protection
